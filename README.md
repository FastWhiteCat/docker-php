# Docker container PHP 7.1 for Magento 2

Included:

- PHP Version 7.1.13 (FPM)
- Composer 1.6.5
- Node.js
- MageRun for M2
- ImageMagic 3.4.3
- PhpRedis 4.1.0 https://github.com/phpredis/phpredis

Libraries
- git
- vim
- wget
- lynx
- psmisc
- bzip2
- cron
- supervisor

## Description

Reusable base container in projects on Magento2.
